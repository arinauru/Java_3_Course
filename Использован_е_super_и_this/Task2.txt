Когда в классе C вызывается метод method(), если переменная a и метод method() 
не определены в классе B, а только в классе A, то при использовании super.a и 
super.method() будет происходить следующее. Переменная a и метод method() будут 
взяты из класса A, потому что класс C наследует их через класс B. В методе C, 
когда вызывается super.a, доступ к полю a осуществляется через суперкласс A, 
и его значение (например, 10) будет присвоено переменной. Вызов super.method() 
приведет к выполнению метода method() из класса A, так как метод не переопределен 
в классе B. В результате выполнения программы будет вызван метод method() из класса A, 
и поле a будет взято из класса A. В методе method() из класса C будет выводиться 
значение поля a, которое равно 5 (если так оно и задано в классе A), 
и информация о том, что метод был вызван из класса C.
Вывод программы
Метод из класса A. Значение a: 5
Метод из класса C. Значение a: 5
