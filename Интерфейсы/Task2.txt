Объясните, какая область видимости у интерфейса, объявленного следующим образом.

public interface Interface {
  ...
}
Интерфейс, объявленный с модификатором public, имеет глобальную область видимости, 
то есть доступен из любого места в программе, при условии, что класс или интерфейс, 
пытающийся получить к нему доступ, импортировал соответствующий пакет.
