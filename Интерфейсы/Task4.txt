Объясните, какая область видимости у интерфейса, объявленного следующим образом.

private interface Interface {
  ...
}
Интерфейс не может быть объявлен с модификатором private на уровне файла, так как 
это недопустимо для топ-левел интерфейсов. Однако вложенный интерфейс внутри класса 
или интерфейса может быть объявлен как private, и тогда он будет доступен только 
внутри содержащего его класса.
