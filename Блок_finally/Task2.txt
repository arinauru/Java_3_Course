Можно ли использовать следующую конструкцию (try только с finally):

try {
  ...
} finally {
  ...
}
Да, конструкция try только с блоком finally вполне допустима в Java. 
Это означает, что код внутри блока try выполняется, и независимо от того, 
возникло ли исключение или нет, блок finally всегда будет выполнен.
