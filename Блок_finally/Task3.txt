Можно ли использовать следующую конструкцию (try с двумя finally):

try {
  ...
} finally {
  ...
} finally {
  ...
}
Нет, конструкция try с двумя блоками finally в Java не допускается и вызовет ошибку 
компиляции. Java позволяет только один блок finally, так как после его выполнения 
выполнение программы должно завершиться, и дальнейшие блоки finally не имеют смысла.
